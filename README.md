# AutoDownloadFirIm

这个项目能帮你简化从 fir.im 上下载 apk 的流程，你只需要点击“开始下载”按钮，自动帮你完成以下操作：

1. 打开下载链接
2. 填入密码
3. 点击网页的“确认”按钮
4. 点击网页上的“下载安装”按钮
5. 调用系统的下载工具下载 apk
6. apk 下载完成后自动弹出安装框
